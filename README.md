# asyncswap/skills

Claude Code plugins and skills by [asyncswap](https://github.com/asyncswap).

## Installation

```
/plugin marketplace add asyncswap/skills
```

## Skills

### Solidity LSP

Language server support for Solidity via [solidity-language-server](https://github.com/asyncswap/solidity-language-server).

```
/plugin install solidity-language-server@asyncswap
```
